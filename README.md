# 暴走男朋友

一个适合部署到 GitHub Pages 的手机优先轻互动恋爱小游戏。

## 当前版本能力

- 手机优先主舞台，人物始终居中可见
- `暴揍模式` / `抚摸模式` 双模式切换
- 点击人物不同身体区域触发不同反馈
- 好感值与爱心值双数值系统
- 爱心值商店、装备、里程碑彩蛋
- 连续乱点同一区域会触发生气和临时锁区
- 支持上传照片贴脸，数据只保存在当前浏览器

## v1.1 已实现重点

- 5 个基础热区：头部、脸部、手部、肩膀、身体敏感区
- 6 种表情状态：普通、开心、害羞、生气、无语、疲惫
- 点击反馈：文本气泡、数值漂浮、动画反馈、表情切换
- 好感值与爱心值双系统并存
- 基础道具可视化：奶茶、鲜花、零食，以及现有整蛊/武器体系
- 热区调试开关，方便后续继续迭代

## 目录

- `index.html`：页面入口与舞台结构
- `css/style.css`：页面样式、热区调试、表情与动画
- `js/app.js`：互动逻辑、数值系统、商店、热区配置、本地存档

## 本地预览

直接双击 `index.html` 即可打开。  
如果想用本地服务预览，也可以在当前目录运行：

```powershell
python -m http.server 8600
```

然后访问：

```text
http://127.0.0.1:8600
```

## 部署到 GitHub Pages

1. 创建一个新的 GitHub 仓库
2. 把本目录文件推送到仓库根目录
3. 打开 GitHub 仓库页面
4. 进入 `Settings` -> `Pages`
5. 在 `Build and deployment` 中选择从当前分支部署
6. 选择 `main` 分支和 `/ (root)` 目录
7. 保存并等待 GitHub 生成访问地址
