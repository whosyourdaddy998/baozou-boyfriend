# 暴走男朋友

一个适合部署到 GitHub Pages 的单页互动小游戏。

## 玩法

- 点击不同道具按钮，触发 Q 版男朋友的表情、台词和连击变化
- 自动记录总互动次数、各道具使用次数、已解锁彩蛋
- 第 10 次解锁委屈脸，第 30 次解锁隐藏吐槽，第 50 次触发特殊结局
- 支持本地上传照片覆盖角色头像，数据只保存在当前浏览器

## 目录

- `index.html`：页面入口
- `css/style.css`：页面样式与动画
- `js/app.js`：互动逻辑、本地存储、彩蛋解锁

## 本地预览

直接双击 `index.html` 就能打开。  
如果想用本地服务预览，也可以在当前目录运行：

```powershell
python -m http.server 8600
```

然后访问：

```text
http://127.0.0.1:8600
```

## 部署到 GitHub Pages

1. 创建一个新的 GitHub 仓库
2. 把本目录文件推送到仓库根目录
3. 打开 GitHub 仓库页面
4. 进入 `Settings` -> `Pages`
5. 在 `Build and deployment` 里选择从当前分支部署
6. 保存后等待 GitHub 生成访问地址
